# VyroEngine Commit Authorship Rules

**Version:** 1.0.0

---

## Rule CA-1: No AI / Tool Co-Authors

Commits must **never** include AI or tool attribution. Specifically forbidden in commit messages:

- `Co-Authored-By: Claude ...`
- `Co-Authored-By: Claude Code ...`
- `Co-Authored-By:` any AI assistant or bot
- `🤖 Generated with ...` footers
- Any "Generated by" / "Created with AI" trailers

The repository contributor list must show **human authors only**.

## Rule CA-2: Author Identity

All commits are authored by the human developer:

```
Gaurav <gauravganesh1214@gmail.com>
```

## Rule CA-3: Clean Commit Footers

Commit footers may contain only:
- `Fixes #<issue>` / `Closes #<issue>`
- `Refs #<issue>`
- `BREAKING CHANGE: <description>`

No tool branding, no AI attribution, no auto-generated signatures.

## Rule CA-4: Verify Before Push

Before pushing, run `git log -1 --format='%an <%ae>%n%n%b'` and confirm:
- Author is the human developer
- No `Co-Authored-By` lines referencing AI/bots
- No generated-with footers
